Welcome to Ionize

Ionize is a Content Management System, based on CodeIgniter Application Development Framework.

Ionize makes the management of a website easy, by focusing on an ergonomic aproach. It provides to web developers a easy to run and to customize solution.

Is Ionize for me ?

Ionize is right for you if:

  • You are a web designer or web coder working on small and middle sized website projects.
  • You want a clear and easy to use Content Management System for your clients.
  • Your clients don't want to edit any HTML code.
  • You want flexible CMS, in which you can add modules.

Ionize is not for you if :

  • You want an "out of the box" website with ready to use templates.
  • You don't know anything about websites.
